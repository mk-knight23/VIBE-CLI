## Core Capabilities

### 8 Primitives

Vibe CLI uses 8 unified primitives instead of 60+ scattered tools:

1. **Ask** - Answer questions, explain code
2. **Code** - Generate, refactor, complete code
3. **Debug** - Fix bugs, trace execution
4. **Test** - Write and run tests
5. **Deploy** - Build, ship, release
6. **Agent** - Run autonomous multi-step tasks
7. **Memory** - Remember project context
8. **Plan** - Design architecture, roadmaps

# VIBE v0.0.1 Feature Inventory

VIBE v0.0.1 implements 120 features focused on professional AI engineering.

## 1. Intelligence & Productivity
- **Asynchronous Command Generation**: Non-blocking command suggestions.
- **AI Fallback System**: Intelligent provider selection and fallback.
- **History-Aware Suggestions**: Command history used for better context.
- **AI Docs & Tests**: Automated generation of comprehensive docs and vitest suites.
- **Vibe Doctor**: Self-healing environment diagnostics and automated fixes.

## 2. Web & UI Generation
- **Stack Scaffolder**: Template-based and AI-driven custom project creation.
- **Project Visualizer**: ASCII trees, dependency mapping, and architectural advice.
- **Live Preview Support**: Integrated dev server management.

## 3. Debugging & Testing
- **Deep Error Analysis**: AI-powered stack trace diagnosis (/debug).
- **Automated Remediation**: One-command bug fixing (/fix).
- **Pattern Refactoring**: Identification and application of modularity patterns (/refactor).
- **Code Sentiment**: AI-driven project health and "vibe" analysis (/mood).

## 4. Cloud & DevOps
- **AI CI/CD**: Custom GitHub Actions and GitLab CI generation (/cicd).
- **AI IaC**: Terraform and CloudFormation generation (/iac).
- **Cloud Advisor**: Architecture reviews and deployment targets (/cloud).
- **Enterprise Readiness**: Compliance and production-readiness checks (/readiness).

## 5. Agentic Core
- **Planner Agent**: Multi-step task decomposition.
- **Executor Agent**: Safe shell and file system operations.
- **Audit Logging**: Traceability for all agent actions (/logs).
- **Cost Analytics**: Real-time token usage and expense tracking (/cost).
- **Model Context Protocol (MCP)**: Native integration for rich context.

## 6. Security & Safety
- **Sandbox Mode**: Isolated execution environment.
- **Checkpoints & Undo**: Deterministic state management.
- **Granular Approvals**: Risk-based human-in-the-loop gates.
