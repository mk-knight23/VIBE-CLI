# Vibe CLI - AI-Powered Coding Assistant

Vibe CLI is a terminal-based AI coding assistant that helps you build faster with AI assistance. It supports multiple providers, dozens of models, and is completely free to use.

## Quick Start

```bash
# Install via curl (macOS/Linux)
curl -fsSL https://raw.githubusercontent.com/mk-knight23/VIBE-CLI/main/install.sh | bash

# Or via npm
npm install -g vibe-ai-cli

# Start using Vibe
vibe help
```

## Features

- **Terminal-First Workflow** - Seamless CLI integration that feels natural
- **Lightning Fast** - Sub-second response times with efficient API calls
- **Privacy First** - Your code stays local with no data collection
- **Multi-Provider Support** - OpenRouter, MegaLLM, AgentRouter, Routeway
- **40+ Models** - GPT-4, Claude, Gemini, Llama, DeepSeek, and more
